# Beauty Salon Bot 🤖💅

Telegram бот для записи на процедуры в салон красоты. Клиенты могут записываться на услуги, выбирать мастера и время. Администраторы имеют доступ к панели управления для управления записями, услугами и мастерами.

## Возможности

### Для клиентов
- 📅 Просмотр доступных услуг
- 👩‍🎨 Выбор мастера
- ⏰ Выбор даты и времени записи
- 📝 Просмотр своих записей
- ❌ Отмена записи

### Для администраторов
- 🔐 Специальная админ-панель
- 📊 Просмотр всех записей
- ✏️ Редактирование записей клиентов
- 🛠️ Управление списком услуг
- 👥 Управление мастерами
- 📢 Рассылка уведомлений клиентам

## Технологии

- Python 3.10+
- python-telegram-bot 20.x
- SQLAlchemy (SQLite)
- GitHub Actions (для бесплатного хостинга)
